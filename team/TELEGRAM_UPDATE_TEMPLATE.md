## Good Telegram Habits

- Keep one topic per problem or file submission.
- Attach screenshots when UI behavior is unclear.
- Mention the exact page, document, or file path.
- Do not send credentials, keys, `.env`, `.env.local`, database passwords, or license secrets.
- Do not ask Codex to overwrite files directly. Ask Codex to review and compare first.
