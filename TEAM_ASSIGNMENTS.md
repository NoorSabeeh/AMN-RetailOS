# Team Assignments

This file defines role-based assignments for Phase T1 — Team Alignment & Work Preparation.

Use role-based language only. Do not use personal names, personal hierarchy labels, or individual ability descriptions.

## Product & Backend Coordinator

### Assigned Responsibilities

- Maintain MVP scope discipline.
- Confirm backend business rule priorities.
- Coordinate documentation and next-phase readiness.
- Review blockers from all roles.
- Keep delayed features out of current implementation planning.

### Files to Read

- `README.md`
- `AGENTS.md`
- `CODEX_STATUS.md`
- `TASKS.md`
- `docs/01_PRODUCT_SCOPE.md`
- `docs/02_MVP_V0_1_SCOPE.md`
- `docs/05_BACKEND_ARCHITECTURE.md`
- `docs/14_API_SERVICE_CONTRACTS_DRAFT.md`
- `docs/19_ACCEPTANCE_CRITERIA.md`

### Deliverables

- MVP scope confirmation note
- Backend B1 readiness checklist
- Consolidated blocker list
- Next-phase recommendation
